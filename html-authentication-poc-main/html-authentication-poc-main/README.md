# Styled HTML Authentication System

This project is a responsive, Bootstrap-powered authentication UI prototype built using plain HTML, CSS, and JavaScript.
It includes a full set of pages (login, registration, password reset) and a dashboard layout with a modern look and feel.

## 📄 Pages Included

- `login.html` — login form with validation and simulated authentication flow
- `register.html` — user registration form with password strength feedback
- `forgot-password.html` — submit your email to get a password reset link
- `reset-password.html` — set a new password with visibility toggles + validation
- `dashboard.html` — dashboard layout with navbar, stats, and quick actions

## ✨ Features

- Bootstrap 5 layout and responsive grid system
- Custom styling via `styles.css` (custom color palette, fonts, shadows, hover effects)
- Accessible markup with focus management and `aria-*` attributes
- Smooth transitions & modern UI components
- Simulated form submission flows with loading states

## 🚀 How to Run

1. Clone or download the repository
2. Open `login.html` in your browser
3. Navigate through the pages using the links and buttons
