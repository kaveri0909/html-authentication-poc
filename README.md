# HTML Authentication POC

This project is a simple HTML-based authentication system.

## Pages Included

- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Features

- Simple login simulation
- Registration page
- Forgot password flow
- Reset password page
- Dashboard page
- Redirections using anchor tags only
- No CSS or JavaScript used

## How to Run

1. Download or clone the repository
2. Open login.html in your browser
3. Navigate through the pages using links
